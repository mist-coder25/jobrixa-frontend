# ⚡ Jobrixa Quick Reference

A developer's guide to the new Jobrixa Design System.

## 🎨 Color Palette
| Name | Hex | Usage |
| :--- | :--- | :--- |
| **Main BG** | `#0A0E27` | Application background |
| **Card BG** | `#0F1419` | Cards, modals, sections |
| **Primary** | `#5B9FFF` | Buttons, active states |
| **Border**  | `#1E293B` | Dividers, card borders |
| **Text 1**  | `#FFFFFF` | Primary headings |
| **Text 2**  | `#A0AEC0` | Secondary/Body |

## 🏗️ New Components
- `<Avatar name="Name" size="small|medium|large" />`
- `<FeedbackModal isOpen={...} onClose={...} />`

## 📏 Spacing
Use the 8px grid system ($8px, 16px, 24px, 32px$).

## 🚀 Common Commands
- **Build**: `npm run build`
- **Lint**: `npm run lint`
- **Dev**: `npm run dev`
